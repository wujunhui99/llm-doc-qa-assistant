# PRODUCT_SENSE.md

## Primary User Value
- Fast answer discovery from personal/project documents.
- Trustworthy responses with traceable sources.
- Smooth multi-turn conversation continuity.
- Strong privacy by default: each user can only access their own documents.
- Flexible scope control: ask against one document (`@doc`) or all owned documents (`@all`).

## Product Principles
- Grounded > fluent but ungrounded.
- Predictable > magical.
- Observable behavior > hidden behavior.
